Langkah Langkah Menginstal Arduino IDE Di Windows 10 💡

Note : Ini khusus windows.

1. Download Sofware Arduino IDE Khusus untuk windows 💡

teman teman bisa mengunduh sofware arduino pada linik disini setelah teman teman mengunjungi link diatas maka tampilannya akan seperti dibawah ini, teman teman pilih yang sudah saya berikan kotak merah itu untuk versi windows download disitu.

2. Pilih justdownload

selanjutnya pilih just download untuk hanya mendownload tanpa berdonasi, jika temen temen ingin berdonasi pilih yang disampingnya yaitu contribute download, lalu tunggu sampai proses download selesai.

3. Install Sofware Arduino IDE nya

setelah proses download selesai temen temen bisa klik 2 kali pada file arduino yang sudah didownload, lalu akan muncul tampilan License Agreement atau Persetujuan Instalasi, klik tombol I Agree untuk memulai install software Arduino IDE.

4. Opsi Instalasi

Pada proses opsi instalasi, centang semua komponenya lalu klik next untuk melanjutnya proses instalasinya.

4. Pemilihan lokasi penyimpanan folder

Jika teman teman ingin merubah lokasi penyimpananya klik browse file, jia lokasinya penyimpanannya sudah sesuai langsung saja teman teman klik tombol install untuk memulai proses instalasi.

5. Proses extrak arduino ide

6. Tampilan utama arduino ide

Ini merupakan gambar tampilan jendela utama arduino IDE, nantinya teman teman akan membuat program disini, Setelah semua proses di atas kamu ikuti, maka Software Arduino (IDE) telah berhasil terinstall. Klik Close untuk menutup dialog window.

Penutupan

pada kesempatan kali ini kita sudah berhasil menginstal arduino IDE, jika teman teman mengikuti langkah langkah diatas dengan benar, maka gue rasa tidak akan ada kendala penginstalan, jika teman teman tidak berhasil silakan komentar dibawah ya atau bertanya disosial media gue. sekian artikel kali ini termakasi sudah berkunjung, sekiranya artikel ini bermanfaat teman teman bisa membagikannya hehe.
